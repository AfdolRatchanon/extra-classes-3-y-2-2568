# 📑 Course Outline: การสร้างเว็บไซต์ (Web Portfolio) ด้วย AI บนสมาร์ทโฟน

| รายการ | รายละเอียด |
|---|---|
| **ระยะเวลา** | 4 วัน (วันละ 1 ชั่วโมง 30 นาที / รวม 6 ชั่วโมง) |
| **กลุ่มเป้าหมาย** | นักเรียนระดับ ปวช. และ ปวส. (สาขา DG, IT, DBT) |
| **รูปแบบการเรียน** | ออนไลน์ 100% (ภาคปฏิบัติรายบุคคล) |
| **เครื่องมือ** | สมาร์ทโฟน, Google Gemini, W3Schools Tryit Editor |

---

## 🎯 วัตถุประสงค์การเรียนรู้ (Learning Objectives)

เมื่อจบหลักสูตรนี้ นักเรียนจะสามารถ:

1. เข้าใจโครงสร้างการทำงานพื้นฐานของเว็บไซต์ (HTML) และการตกแต่ง (CSS)
2. อ่านและทำความเข้าใจ Prompt ที่อาจารย์เตรียมไว้ และนำไปปรับแก้ด้วยตนเองได้
3. สามารถทดสอบ (Run) และแก้ไขปัญหาโค้ด (Debug) ผ่านเบราว์เซอร์บนสมาร์ทโฟนได้
4. สร้างและนำเสนอ **Web Portfolio** ของตนเองที่รองรับการแสดงผลบนมือถือ (Responsive) ได้

---

## 🔄 รูปแบบการเรียนแต่ละวัน

แต่ละวันประกอบด้วย 3 ส่วนหลัก:

1. **Theory** — ทฤษฎี HTML/CSS และแนวคิดที่เกี่ยวข้องกับงานวันนั้น
2. **Prompt Study** — ศึกษา Template Prompt ที่อาจารย์เตรียมไว้ เข้าใจว่าแต่ละส่วนทำหน้าที่อะไร
3. **Workshop** — นำ Prompt ไปใช้กับ Gemini แล้วปรับแก้ Prompt ด้วยตนเองให้ตรงกับข้อมูลของตัวเอง

**การส่งงานทุกวัน:** Source Code + ภาพการแสดงผล → Google Classroom

## 📅 โครงสร้างเนื้อหา (Course Syllabus)

### 📌 [วันที่ 1: รู้จักโครงสร้างเว็บ & สร้าง Portfolio ครั้งแรก](/day1) (1.30 ชม.)

> **เป้าหมาย:** นักเรียนอ่านโค้ด HTML เบื้องต้นได้ และสั่ง AI สร้าง Portfolio โครงสร้างพื้นฐาน

- **[Theory]** โครงสร้าง HTML, Tags พื้นฐาน, Internal CSS
- **[Prompt Study]** สูตร Prompt: บทบาท + ความต้องการ + รายละเอียด + ข้อจำกัด
- **[Workshop 1]** ฝึก 5 Prompts — จาก Template → ดัดแปลง → อิสระ
- **ส่งงาน:** Source Code + ภาพการแสดงผล

### 📌 [วันที่ 2: CSS & การใส่ข้อมูลจริง](/day2) (1.30 ชม.)

> **เป้าหมาย:** นักเรียนเข้าใจ CSS เพิ่มขึ้น และ Gen โค้ดใหม่ที่ใส่ข้อมูลจริงพร้อม Style ที่ดีขึ้น

- **[Theory]** CSS Properties, Responsive Design, Flexbox
- **[Prompt Study]** เปรียบเทียบ Prompt วันที่ 1 กับ 2
- **[Workshop 2]** ฝึก 5 Prompts — เพิ่ม Style + ข้อมูลจริงของตนเอง
- **ส่งงาน:** Source Code + ภาพการแสดงผล

### 📌 [วันที่ 3: รูปภาพ ลิงก์ และ Portfolio ฉบับสมบูรณ์](/day3) (1.30 ชม.)

> **เป้าหมาย:** นักเรียน Gen Portfolio ฉบับสมบูรณ์พร้อมรูปภาพและลิงก์ติดต่อ

- **[Theory]** แท็ก `<img>`, แท็ก `<a>`, Emoji Icon
- **[Prompt Study]** วิเคราะห์ Prompt ฉบับสมบูรณ์
- **[Workshop 3]** ฝึก 5 Prompts — รูปภาพ + ลิงก์ + Portfolio สมบูรณ์
- **ส่งงาน:** Source Code + ภาพการแสดงผล

### 📌 [วันที่ 4: ประเมินผลความรู้](/day4) (1.30 ชม.)

> **เป้าหมาย:** ทดสอบความเข้าใจทั้งภาคทฤษฎีและแนวคิดการประยุกต์ใช้ AI

| เวลา | กิจกรรม |
|---|---|
| 10 นาที | เช็คชื่อและชี้แจงกติกาการสอบ |
| 80 นาที | สอบปรนัย (Multiple-Choice) ผ่าน Google Forms จำนวน 30–40 ข้อ |

**ขอบเขตเนื้อหาที่สอบ:**

| หัวข้อ | สัดส่วน |
|---|---|
| คำศัพท์และโครงสร้าง HTML/CSS เบื้องต้น | 30% |
| การออกแบบเพื่อรองรับหน้าจอมือถือ (Responsive Design) | 20% |
| หลักการเขียนคำสั่ง (Prompt Engineering) และการใช้งาน AI | 30% |
| ความเข้าใจด้าน UI/UX และการนำเสนอข้อมูลบนเว็บไซต์ | 20% |
