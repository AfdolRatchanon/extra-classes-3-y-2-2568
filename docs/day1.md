## 📖 ทฤษฎี — โครงสร้างเว็บไซต์ (HTML)

### เว็บไซต์คืออะไร?

เว็บไซต์คือไฟล์ข้อความธรรมดาที่เบราว์เซอร์ (Chrome, Safari) แปลงเป็นหน้าเว็บสวยงาม
ภาษาที่ใช้คือ **HTML** (HyperText Markup Language)

### โครงสร้างพื้นฐานของ HTML

```html
<!DOCTYPE html>
<html lang="th">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ชื่อเว็บ</title>
    <style>
      /* CSS อยู่ที่นี่ */
    </style>
  </head>
  <body>
    <!-- เนื้อหาเว็บอยู่ที่นี่ -->
  </body>
</html>
```

### Tags ที่ต้องรู้

| Tag | หน้าที่ | ตัวอย่าง |
|---|---|---|
| `<h1>` – `<h3>` | หัวข้อ (ใหญ่→เล็ก) | `<h1>ชื่อของฉัน</h1>` |
| `<p>` | ย่อหน้าข้อความ | `<p>เกี่ยวกับฉัน...</p>` |
| `<div>` | กล่องจัดกลุ่ม | `<div>...</div>` |
| `<style>` | ใส่ CSS ภายใน | อยู่ใน `<head>` |
| `<span>` | จัดกลุ่มข้อความ | `<span>ทักษะ</span>` |

### Internal CSS คืออะไร?

CSS คือการตกแต่ง เช่น สี ขนาด ตำแหน่ง
**Internal CSS** หมายถึงเขียน CSS ไว้ในไฟล์เดียวกับ HTML ภายใน `<style>`

```html
<style>
  body {
    background-color: #f0f4ff;
    font-family: sans-serif;
  }
  h1 {
    color: #1a73e8;
  }
</style>
```

::: tip ทำไมต้องใช้ Internal CSS?
เพราะเราสร้างไฟล์เดียวใน W3Schools — ไม่ต้องสร้างไฟล์หลายไฟล์
:::

## 🧠 Prompt Study — เรียนรู้สูตร Prompt

### Prompt คืออะไร?

Prompt คือ "คำสั่ง" ที่เราพิมพ์ให้ AI (Gemini) ทำตาม
ยิ่ง Prompt ชัดเจน AI ก็ยิ่งสร้างโค้ดได้ตรงที่เราต้องการ

### สูตร Prompt ที่ดี

```
[บทบาท] + [ความต้องการ] + [รายละเอียด] + [ข้อจำกัด]
```

| ส่วน | หมายถึง | ตัวอย่าง |
|---|---|---|
| **บทบาท** | บอก AI ว่าให้เป็นใคร | "คุณคือนักพัฒนาเว็บมืออาชีพ" |
| **ความต้องการ** | บอกว่าต้องการอะไร | "สร้างหน้าเว็บ Portfolio" |
| **รายละเอียด** | บอก Section ที่ต้องการ | "มีส่วน: ชื่อ, เกี่ยวกับฉัน, ทักษะ" |
| **ข้อจำกัด** | บอกข้อบังคับ | "ใช้ HTML + Internal CSS เท่านั้น รองรับมือถือ" |

::: warning สังเกตให้ดี
Prompt ที่ดี ≠ Prompt ที่ยาว
แต่ต้องครบทั้ง 4 ส่วน และชัดเจน
:::

### 🟢 Prompt 1 — สร้าง Portfolio โครงสร้างพื้นฐาน
**(อาจารย์เตรียมให้ — ใช้ได้เลย)**

::: info คัดลอก Prompt นี้ไปวางใน Gemini
```
คุณคือนักพัฒนาเว็บมืออาชีพ สร้างหน้าเว็บ Portfolio ส่วนตัวสำหรับนักเรียน ปวช./ปวส. โดยใช้ HTML และ Internal CSS เท่านั้น (ในไฟล์เดียว) ต้องมีส่วนประกอบดังนี้: 1) ส่วนหัว (Header) แสดงชื่อและตำแหน่ง 2) ส่วน "เกี่ยวกับฉัน" 3) ส่วน "ความสนใจ" ใช้ธีมสีฟ้า-ขาว ข้อความภาษาไทย และต้องรองรับหน้าจอมือถือด้วย meta viewport
```
:::

**หลังรัน สังเกตดูว่า:**
- [ ] มี Header แสดงชื่อไหม?
- [ ] มี Section เกี่ยวกับฉันไหม?
- [ ] มี Section ความสนใจไหม?
- [ ] สีธีมเป็นฟ้า-ขาวไหม?
