## 📝 สรุปเนื้อหาสำหรับทบทวน

### 1. HTML Tags ที่ต้องรู้

| Tag | หน้าที่ |
|---|---|
| `<!DOCTYPE html>` | ประกาศเอกสาร HTML5 |
| `<html>` | รากของเอกสาร |
| `<head>` | ส่วน metadata (ไม่แสดงบนหน้าเว็บ) |
| `<body>` | ส่วนเนื้อหาที่แสดงบนหน้าเว็บ |
| `<h1>`–`<h3>` | หัวข้อ ลำดับ 1–3 |
| `<p>` | ย่อหน้าข้อความ |
| `<div>` | กล่องจัดกลุ่มเนื้อหา |
| `<img>` | แสดงรูปภาพ |
| `<a>` | สร้างลิงก์ |
| `<style>` | ใส่ CSS ภายในไฟล์ (Internal CSS) |
| `<span>` | จัดกลุ่มข้อความ inline |
| `<meta>` | ข้อมูล metadata เช่น viewport, charset |

### 2. CSS Properties สำคัญ

| Property | หน้าที่ | ตัวอย่าง |
|---|---|---|
| `color` | สีตัวอักษร | `color: #333;` |
| `background-color` | สีพื้นหลัง | `background-color: #fff;` |
| `font-family` | ฟอนต์ | `font-family: Sarabun;` |
| `font-size` | ขนาดตัวอักษร | `font-size: 16px;` |
| `padding` | ระยะห่างข้างใน | `padding: 16px;` |
| `margin` | ระยะห่างข้างนอก | `margin: 0 auto;` |
| `border-radius` | มุมโค้ง | `border-radius: 8px;` |
| `max-width` | ความกว้างสูงสุด | `max-width: 600px;` |
| `display: flex` | จัดวางแบบ Flexbox | `display: flex;` |
| `text-decoration` | ตกแต่งข้อความ | `text-decoration: none;` |

### 3. Responsive Design

- **meta viewport** — บอกมือถือให้แสดงผลขนาดจริง
  ```html
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  ```
- **max-width + width: 100%** — ป้องกันเนื้อหาล้นหน้าจอ
- **box-sizing: border-box** — คำนวณ padding รวมใน width

### 4. Prompt Engineering

**สูตร 4 ส่วน:**

| ส่วน | คำถามที่ตอบ |
|---|---|
| **บทบาท** | AI ควรเป็นใคร? |
| **ความต้องการ** | ต้องการอะไร? |
| **รายละเอียด** | มีรายละเอียดอะไร? |
| **ข้อจำกัด** | ห้ามทำอะไร / ต้องทำอะไร? |

**Iterative Prompting** = การปรับแก้ Prompt ต่อเนื่องจากผลลัพธ์ก่อนหน้า

### 5. UI/UX เบื้องต้น

- **UI (User Interface)** = รูปลักษณ์ภายนอกของเว็บ (สี, ฟอนต์, Layout)
- **UX (User Experience)** = ประสบการณ์การใช้งาน (ใช้ง่าย, ข้อมูลชัดเจน)
- **Portfolio ที่ดี** ต้องมีข้อมูลครบ, อ่านง่าย, แสดงผลดีบนมือถือ
